# Checkmater Privacy Policy

Effective date: March 27, 2026

This Privacy Policy applies to the Checkmater plugin for JetBrains products ("Checkmater", "the Plugin").

## 1. Summary

Checkmater is a local JetBrains plugin for chess puzzles and local play against a bot.

The Plugin does not require an online account and does not provide internet multiplayer features.

## 2. Data Stored Locally

Based on the current codebase, the Plugin stores the following local data:

- plugin settings and gameplay progress, including preferred puzzle mode, solved levels, streak, best streak, and total attempts.

Local settings are stored by the JetBrains platform in the IDE configuration environment.

## 3. Data Sent Over Network

The Plugin is intended to work locally for puzzle play and bot play.

This Privacy Policy is written for the local plugin use case described above. It does not describe any separate website or backend service, because internet gameplay is not part of this plugin.

## 4. What Checkmater Does Not Claim

Checkmater does not claim to collect analytics, advertising identifiers, or telemetry in the currently verified code paths reviewed for this policy.

This statement is limited to the code verified in the repository as of March 27, 2026. If the Plugin is changed later, this policy should be updated accordingly.

## 5. No Online Account Requirement

The Plugin does not require a user account to use its local puzzle and bot features.

## 6. Legal Basis and User Choice

Use of the Plugin is optional and user-initiated.

If you do not want local plugin data such as settings or gameplay progress to remain on your machine, you should remove the Plugin and delete the related IDE configuration data.

## 7. Data Security

Checkmater relies on local JetBrains platform storage for plugin state. No security method is guaranteed to be perfect, and use of the Plugin is at the user's own risk.

## 8. Data Retention

Local settings remain on the user's machine until removed by the user, the IDE, or normal configuration cleanup.

## 9. Children's Privacy

Checkmater is not directed specifically to children.

## 10. International Use

The Plugin is used locally inside JetBrains products. This policy does not describe cross-border backend processing because internet services are not part of this plugin.

## 11. Changes to This Policy

This Privacy Policy may be updated when the Plugin or its data practices change. The effective date above should also be updated when changes are made.
